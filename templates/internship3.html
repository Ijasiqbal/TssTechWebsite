Position Qualifications

Software tester
    • Pursuing a degree in Computer Science, Software Engineering, or a related field.
    • Familiarity with software testing concepts and methodologies is a plus.
    • Familiarity with version control systems (e.g., Git) and development tools (e.g., IDEs, debuggers).
    • Ability to work independently and collaboratively in a team environment.
    • Strong problem-solving skills ,interpersonal skills and attention to detail.

    • Responsibilities

    • test software applications or systems to identify bugs, errors, and other issues.
    • Assist in creating test plans and strategies for different aspects of the software, such as functionality, performance, security, and usability.
    • Execute test cases and scripts to verify the functionality and performance of software applications.
    • Stay updated with the latest testing methodologies, tools, and technologies to improve your testing skills and contribute effectively to the team.
    • Work closely with software developers, project managers, and other team members to ensure timely resolution of issues and delivery of high-quality software.
    • Document and report bugs and issues found during testing accurately. This includes providing detailed descriptions and steps to reproduce the bugs.
